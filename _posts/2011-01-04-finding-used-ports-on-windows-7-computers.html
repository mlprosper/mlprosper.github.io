Recently while troubleshooting a WCF service on my computer I needed to find out which process was using port 33333.  After much searching and headache I found that using this command gave me a list of ports and the corresponding ID of the process using that port:

netstat -o -n -a

You can then use the ID in task manager to find the process name and kill it.

You might have to go to View-->Select Columns to show the PID column.

In my case it turned out that PGPTray was using the port!
