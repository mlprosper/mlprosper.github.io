By default IIS 7.5 won’t serve .less files to the client.  In order to change that, I needed to set a MIME type in IIS Manager.  First open up IIS Manager, and click the node for your web application. Choose MIME Types:

Then add a type for .less files:

Click OK and Voila! You now have .less files treated like .css!

To do it via the command line, please see this post:
